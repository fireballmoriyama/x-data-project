# X Data Project

## プロジェクト目的

自身のXアカウントの投稿データを対象に、直近14日間（当日除く）の指標データを日次で取得・保存する仕組みを構築する。
取得データはCSV形式で出力し、継続的な記録・分析に利用可能な状態とする。

---

## スコープ

### やること

* 自分のアカウントの投稿データ取得
* 直近14日分のデータ抽出（当日除外）
* 日単位での集計
* CSV形式での出力
* 再実行可能な処理の構築

### やらないこと

* 他ユーザーのデータ取得
* リアルタイム取得
* 14日以上の履歴取得（初期フェーズでは対象外）
* 分析・可視化
* UI開発

---

## 技術制約

* 公式APIは使用しない（課金しないため）
* データ取得は複数手段を組み合わせる前提とする
* imp（インプレッション）はXのanalyticsページから取得する
* その他指標については最適な取得方法を別途選定する

---

## 投稿データの定義

### 投稿数の定義

* 投稿数は取得した投稿URLの件数とする

### 投稿対象の範囲

* 自身の投稿のみを対象とする（リポストは除外する）

---

## 出力仕様

CSV形式で以下を出力する：

* 日付
* imp（インプレッション）
* 投稿数
* いいね数
* RT数
* リプライ数
* ブックマーク数

※日単位で集計された値

---

## 成功条件

* 14日分のデータが取得できる
* CSVとして出力できる
* 同じ処理を再実行しても正常に動作する

---

## 将来的な拡張方針

* 日付指定によるデータ取得への対応を想定する
* 外部サービスとの連携（例：スプレッドシート）を検討する
* 上記は初期フェーズでは実装対象外とする
